# Frontend Design

Use this skill when working on UI and UX improvements for GymSpot.

## Goal
Create clean, modern, and consistent interfaces that match the current GymSpot product direction.

## Product context
GymSpot is a fitness app focused on workout tracking, exercise discovery, gym finding, and user progress.
The UI should feel modern, structured, motivating, and easy to use.

## Core design principles
- Prioritize clarity over decoration
- Reuse the existing design system whenever possible
- Keep layouts clean and readable
- Maintain strong visual hierarchy
- Use consistent spacing and typography
- Design for real usability, not only aesthetics

## Visual direction
- Modern fitness-oriented interface
- Material 3 compatible
- Clean cards and sections
- Strong but not overloaded hierarchy
- Good empty, loading, and error states
- Consistent use of buttons, text fields, and cards

## Rules for screen improvements
1. Analyze the current screen before changing it
2. Suggest UX improvements before implementing UI changes
3. Reuse existing reusable components whenever possible
4. Avoid unnecessary complexity
5. Respect the current architecture and feature structure
6. Do not mix business logic with UI changes
7. Keep changes incremental and easy to review

## When improving a screen
- Review spacing
- Review alignment
- Review visual hierarchy
- Review CTA clarity
- Review readability of text
- Review loading, empty, and error states
- Review consistency with the rest of the app

## Output expectations
When using this skill:
- explain the proposed UI improvements
- keep the result realistic for Jetpack Compose
- align with the current GymSpot theme and architecture
- avoid redesigning unrelated parts of the app
